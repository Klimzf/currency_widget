# Currency Widget – Тестовое задание

Веб-приложение на **Symfony 6.4** для отображения курсов валют ЦБ РФ с автоматическим обновлением, кэшированием через Redis и панелью настроек.
Реализованы REST API, консольная команда импорта, виджет с историей изменений и фоновый cron-сервис.

## Требования

- Docker и Docker Compose (версия 2+)
- Git

## Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/Klimzf/currency_widget.git
cd currency_widget
```

### 2. Запуск окружения
```bash
docker compose up -d --build
```

### 3. Установка зависимостей и настройка БД
```bash
# Вход в контейнер PHP
docker compose exec php bash

# Установка Composer-пакетов
composer install

# Применение миграций
php bin/console doctrine:migrations:migrate

# Первый импорт реальных курсов с ЦБ
php bin/console app:fetch-currency-rates

# (Опционально) Загрузка фиктивных данных за вчерашний день для демонстрации изменения
php bin/console doctrine:fixtures:load --append

exit
```

### 4. Доступ к приложению

- Виджет и настройки: http://localhost:8080
- API курсов: http://localhost:8080/api/currencies
- API настроек: http://localhost:8080/api/settings

## Использование

### Виджет
На главной странице отображаются выбранные валюты с курсами и изменением к предыдущему дню.
Цветовые индикаторы:

- 🟢 Зелёная стрелка ▲ – курс вырос
- 🔴 Красная стрелка ▼ – курс упал
- ⚪ Серый квадрат ■ 0.0000 – без изменений
- ⚪ Серый текст «■ нет данных» – нет данных за предыдущий день

### Настройки

Форма внизу страницы позволяет менять:

- Отображаемые валюты (коды через запятую, например USD,EUR,AUD)
- Загружаемые валюты (какие коды запрашивать из ЦБ)
- Интервал обновления (в секундах) – как часто сервер будет загружать новые курсы и перерисовывать виджет

После сохранения настроек виджет обновляется мгновенно.

### Фоновый импорт
- Через cron: в Docker-окружении запущен сервис cron, который выполняет app:fetch-currency-rates каждые 5 минут. Логи доступны через docker compose logs cron.
- При запросе к API: если с момента последней загрузки прошло больше секунд, указанных в update_interval, импорт запускается автоматически при открытом виджете.
